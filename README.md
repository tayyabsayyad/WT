# WT
Wireless Technology

Instructions to use 

1. Install mkdocs using pip
    $pip install mkdocs
    
2. git clone https://github.com/tayyabsayyad/WT.git

    
3. cd WT

4. mkdocs serve

5. You will get following output, clicl on the link http://127.0.0.1:8000/
